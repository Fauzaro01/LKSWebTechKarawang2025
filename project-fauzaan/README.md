# Facegram - Personal Learning Project

Folder ini berisi hasil pengerjaan saya selama proses belajar dan latihan mengerjakan soal **LKS Web Technologies Kabupaten Karawang 2025**.

## 🎯 Tujuan

Project ini dibuat sebagai:

* Dokumentasi proses belajar Laravel dan Frontend Development.
* Simulasi pengerjaan soal LKS Web Technologies.
* Portofolio pengembangan aplikasi web.

## 📂 Struktur

```text
project-fauzaan/
├── facegram-backend/
└── facegram-frontend/
```

## 🛠️ Teknologi

### Backend

* Laravel
* Laravel Sanctum
* MySQL
* REST API

### Frontend

* Vue.js
* Vue Router
* Axios
* Bootstrap
* AdminLTE

Beberapa fitur:

* Masih memiliki bug.

Namun struktur kode sengaja dibuat agar lebih mudah dipahami oleh pembelajar dibandingkan hasil generasi AI.

## 🚀 Menjalankan Project

### Backend

```bash
cd facegram-backend

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```

### Frontend

```bash
cd facegram-frontend

npm install

npm run dev
```
