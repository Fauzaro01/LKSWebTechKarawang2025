# 🏆 LKS Web Technologies Kabupaten Karawang 2025

Repository ini berisi dokumentasi, sumber belajar, template, dan hasil pengerjaan saya untuk **Lomba Kompetensi Siswa (LKS) SMK Bidang Web Technologies Tingkat Kabupaten Karawang Tahun 2025**.

Tujuan repository ini adalah:

* Menjadi dokumentasi perjalanan belajar saya.
* Menjadi portofolio pengembangan aplikasi web menggunakan Laravel dan React.js.
* Menjadi referensi bagi peserta LKS Web Technologies berikutnya.
* Menyediakan template dan bahan latihan berdasarkan soal yang digunakan saat lomba.

---

## 📋 Tentang Soal

Pada LKS Kabupaten Karawang 2025, peserta diminta membuat aplikasi media sosial sederhana bernama **Facegram**.

Fitur utama yang harus dibuat meliputi:

<details>
  <summary><b>Backend (REST API)</b></summary>

- Authentication (Register, Login, Logout)
- Create Post
- Delete Post
- Get Posts
- Follow User
- Unfollow User
- Get Following
- Accept Follow Request
- Get Followers
- Get Users
- Get User Detail

</details>

<details>
  <summary><b>Frontend</b></summary>

- Register Page
- Login Page
- Homepage (News Feed)
- Explore People
- Follow Requests
- User Profile
- Create New Post

</details>

#### Framework yang digunakan pada soal:
- Laravel
- React.js
